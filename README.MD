Dictionary App
A simple and responsive dictionary web app that lets users search English words and view definitions, pronunciations, and audio. Includes accordion sections for extra language resources.

Technologies Used
-HTML, CSS, JavaScript
-FontAwesome Icons
-dictionaryapi.dev API

What I Did
-Integrated dictionary API for word lookup
-Displayed word, definition, and pronunciation audio
-Built interactive accordion sections with toggle functionality

JavaScript Structure
script.js: Fetches word data and updates the UI
accordion.js: Controls accordion toggle behavior
