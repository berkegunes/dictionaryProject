<!-- Advanced Dictionary -->

<!-- Overview -->

The Advanced Dictionary project is a web-based dictionary application that allows users to search for English words and retrieve their meanings, along with pronunciation audio. The project also features an interactive accordion section for additional language tools.

<!-- Features -->

-Search for English words and retrieve meanings via an external API.

-Display pronunciation audio for searched words.

-Interactive accordion sections for additional dictionary-related tools.

-Responsive design with a modern UI.

<!-- Technologies Used -->

-HTML5: Structuring the web page.

-CSS3: Styling and layout.

-JavaScript (Vanilla JS): Implementing search functionality and interactive elements.

-API: Fetching dictionary data from https://api.dictionaryapi.dev.

-Font Awesome: Icons used for UI elements.

<!-- Scripts -->
<!-- script.js -->

-Fetches word definitions using the Dictionary API.

-Displays the retrieved word meaning and pronunciation.

-Handles error cases when the word is not found.

<!-- accordion.js -->

-Toggles visibility of accordion sections.

-Changes icon states dynamically.

Styles

-The styles are defined in style.css using CSS variables and modern responsive design techniques.
